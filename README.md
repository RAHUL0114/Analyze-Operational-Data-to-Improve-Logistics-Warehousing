# Analyze-Operational-Data-to-Improve-Logistics-Warehousing
Analyze Operational Data to Improve Logistics & Warehousing Efficiency is an end-to-end data analytics and machine learning project focused on improving logistics performance using a real-world operational dataset. The workflow includes data cleaning in Python (VS Code), storing the processed dataset in MySQL, and performing SQL-based analysis and exploratory data analysis (EDA) to answer key business questions and identify operational KPIs. Insights derived from the analysis are visualized through an interactive Power BI dashboard. Additionally, machine learning models such as Logistic Regression and KNN Classifier are implemented to predict delivery accuracy (whether deliveries are on time or not), and model performance is compared to determine the most effective approach.
